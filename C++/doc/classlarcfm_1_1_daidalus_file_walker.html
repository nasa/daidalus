void larcfm::DaidalusFileWalker::selectTraffic ( const std::string &  name)

By default all aircraft that are not the ownship are considered to be traffic. This method add a particular aircraft to the list of selected aircraft. Several aircraft can be selected, but if the list of selected aircraft is non empty, only the aircraft in the list are considered traffic.
